# Instant Roster

A rostering platform focused on staff satisfaction, preferred hours, availability, skills, fairness, overtime preference, and goal achievement percentage.

## Current principles

- Staff satisfaction first
- No wages or payroll calculations yet
- Web app online from the beginning
- GitHub-backed from the beginning
- Vercel auto-deploy from GitHub
- GitHub Actions CI checks
- Supabase/Postgres backend planned
- Python OR-Tools optimiser planned
- Expo mobile app planned later
